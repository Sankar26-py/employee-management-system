# Employee Management System

A production-style Django REST Framework project demonstrating advanced PostgreSQL ORM queries.

---

## Tech Stack

- Python 3.12
- Django 5.2
- Django REST Framework
- PostgreSQL
- Faker
- Pytest

---

## Features

### Employee Management

- Departments
- Employees
- Projects
- Tasks

### Advanced Django ORM

- select_related
- prefetch_related
- annotate
- aggregate
- Subquery
- Exists
- Window Functions
- Case / When
- F Expressions
- Custom Managers

### PostgreSQL Features

- Row Locking
- Query Plans
- Window Functions
